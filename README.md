# manual-java-spring
# De lo básico de Java a Spring 

Este repositorio contiene un manual en LaTeX y ejemplos de código práctico orientados a hispanohablantes. El objetivo es documentar el aprendizaje y desarrollo desde los fundamentos de Java hasta la creación de APIs con Spring Boot.

## Estructura del Proyecto
* `/docs-latex`: Código fuente del manual y anotaciones teóricas.
* `/codigo-java`: Ejercicios, clases y proyectos prácticos.

**Próximos pasos en el desarrollo (2-3 años):** Depues de concluir esta manual se hablara de una implementacion con BD, Microcontroladores ESP32, C++, Protocolos HTPP y MQTT, este manual solo es el punto de partida para lo antes mencionado.
